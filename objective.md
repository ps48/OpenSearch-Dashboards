We have added in a new feature in OSD called datasets, these are nothing but extended index patterns with a signal type: logs, trace or metrics. This is part of dataset_management plugin.

Need your help to develop the new correlations feature. 
1. In trace dataset, we will have a new tab called correlated datasets which will show the trace to log correlation. A correlation object can only be edited/created in the trace dataset correlated datasets tab. 
2. In logs dataset, we will have a new tab called correlated traces which will show the datasets it is part of. A correlation object can only be view in the logs dataset's correlated traces tab. 

A trace dataset can only be part of 1 correlation 
A trace dataset can only have 5 logs dataset for correlations

logs correlation tabs ~/Desktop/logs-dataset-correlation-mocks
traces correlation tabs ~/Desktop/trace-dataset-correlation-mocks
